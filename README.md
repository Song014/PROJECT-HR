# 프로젝트 임포트시 해야할일
* src/main/resources 파일내 db.properties, email.properties 파일 생성
* db.properties
    ```
    driver=oracle.jdbc.driver.OracleDriver
    url={DB 경로}
    username={아이디}
    password={비밀번호}
    ```
* email.properties
    ```
    id={네이버 아이디}
    pwd={네이버 비밀번호}
    ```
